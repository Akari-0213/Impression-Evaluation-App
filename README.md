# Impression-Evaluation-App

PythonライブラリPsychoPy（サイコパイ）を用いた心理学実験用のアプリケーション

## 概要 / 機能

文章が記載された画像を提示し、被験者に5件法で印象評価を行ってもらうアプリ

- **5件法の評価項目**:
  - `a`: [するどい(1) - まるい(5)]
  - `b`: [かたい(1) - やわらかい(5)]
  - `c`: [ちかよりがたい(1) - したしみやすい(5)]
  - `d`: [こうげきてき(1) - やさしい(5)]
  - `e`: [ネガティブ(1) - ポジティブ(5)]
- **画面遷移**: 各文（全36文）についてすべての項目を選択すると、自動的に次の文章へ画面遷移します。
- **全体評価**: 個別の文章だけでなく、文章全体に対する印象評価も同様に行います。
- **データ保存**: すべての入力が完了すると、被験者番号とともに結果がCSVファイルとして自動保存されます。

---

## セットアップ（初回のみ）

以下の手順は環境構築のため初回のみ必要です。一度実行すれば、次回以降は行う必要はありません。

1. **Pythonのインストール**
   [Python 3.11.9 ダウンロードページ](https://www.python.org/downloads/release/python-3119/) にアクセスし、「Windows installer (64-bit)」をダウンロードしてインストールしてください。

2. **依存ライブラリのインストール**
   同梱されている `install.bat` をダブルクリックして実行してください。

---

## 使い方（実験の実施）

1. **アプリの起動**
   `start_app` （または `start_app.bat`）をダブルクリックしてください。アプリが起動します。
   
2. **実験の中止について**
   途中で実験を中止したい場合は、アプリの裏側で開いている黒いウィンドウ（コマンドプロンプト）を閉じてください。
   *※正常に最後まで進める場合は、実験が終わるまでこのウィンドウを絶対に閉じないでください。*

3. **実験の終了**
   すべての評価が終わると、ウィンドウは自動的に閉じられます。

---

## 出力データについて

実験結果は `data` フォルダ内に `result_〇_impression.csv` として情報が追加・保存されます。

### CSVカラム（列）の読み方

- **`first_all` / `end_all`**:
  文章全体の印象評価の結果です。
- **`1_`, `2_`, `3_`, ...**:
  1文ずつの印象評価結果です。`settings` フォルダ内の `○○_images` フォルダにある画像番号と対応しています。
- **サフィックス（`a` 〜 `e`）**:
  印象対の項目を表しています。
  - `a`: [するどい(1) - まるい(5)]
  - `b`: [かたい(1) - やわらかい(5)]
  - `c`: [ちかよりがたい(1) - したしみやすい(5)]
  - `d`: [こうげきてき(1) - やさしい(5)]
  - `e`: [ネガティブ(1) - ポジティブ(5)]

**【データ例】**
列名が `2_c` の場合、「本文2文目の画像」に対する「`c` = [ちかよりがたい(1) - したしみやすい(5)]」の評価結果となります。
